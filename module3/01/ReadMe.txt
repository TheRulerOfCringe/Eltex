Получаем аргументы
Половину оставим в родительском процессе, половину перекинем на дочерний
Аргумент надо так разбирать:
Если int, то считать, как int. Вывести его и х2 его
Если float, то считать, как float. Вывести его и х2 его
Иначе просто аргумент принтим

argv = program, 1, 2, 3
argc - 1 = количество переданных параметров
mid = (argc)/2 = половина аргументов

for (int i = 1, i < mid, i++)
{
    printf(); // для родителя
}

for (int i = mid, i < argv - 1, i++)
{
    printf(); // для ребёнка
}
